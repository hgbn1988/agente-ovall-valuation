Base de Conhecimento

RAG — Retrieval Augmented Generation / Vector Database
A memória do agente: tudo que ele já aprendeu e guardou sobre imóveis, leis e mercado
O que é Um banco de dados inteligente que armazena documentos, transações históricas, legislação urbanística e premissas macroeconômicas em formato vetorial.
Como funciona Quando o agente recebe uma pergunta, ele busca nos documentos mais relevantes usando similaridade semântica — como um Google interno ultra-preciso que entende contexto, não apenas palavras-chave.
Por que existe O LLM sozinho "inventa" dados (alucinação). O RAG obriga o agente a usar apenas informações reais, validadas e rastreáveis. É a diferença entre chutar e saber.

Orquestrador

Orchestration Layer — LLM + Deterministic Math Engine + Task Planner
O maestro: decide o que fazer, em que ordem, e com qual ferramenta — sem nunca errar a conta
O que é O cérebro central que coordena todo o processo de valuation, combinando raciocínio de IA com cálculo matemático determinístico.
Como funciona O LLM (GPT-5.4, Claude Opus 4.6 ou Gemini 3.1 Pro) raciocina e toma decisões. O Motor Matemático (Python Decimal com 28 dígitos de precisão) calcula sem margem de erro. O Planejador decompõe a tarefa em sub-tarefas e aciona os agentes especializados.
Por que existe Um LLM sozinho erra matemática — é um fato. Separar o raciocínio (IA) do cálculo (código determinístico) elimina 100% das alucinações numéricas. É como ter um CEO que pensa e um CFO que calcula.

O que é Cinco agentes menores e independentes, cada um especializado em uma área do valuation. Nomeados com elementos da Tabela Periódica — identidade Ovall.
Como funciona O Orquestrador delega tarefas específicas. Cada agente tem seu próprio prompt de sistema (persona de especialista), ferramentas dedicadas e acesso à base de dados relevante. Trabalham em paralelo quando possível.
Por que existe Um generalista erra detalhes. Um agente que tenta fazer tudo ao mesmo tempo perde profundidade. Especialistas garantem qualidade "nível PhD" em cada área — a mesma lógica de um escritório com sócios especializados.

Tripla Validação

Triple Validation Pipeline — Deterministic + AI Audit + Cross-Reconciliation
O selo de garantia: nada sai sem passar por 3 conferências independentes — como um tribunal com 3 juízes
O que é Um pipeline de três camadas de verificação sequencial. Cada camada usa um método diferente para garantir que o resultado esteja correto.
Por que existe Em finanças imobiliárias, um erro de vírgula pode custar milhões. A tripla validação é o padrão ouro: se três métodos independentes concordam, o resultado é confiável.
1
Matemática Determinística
Python Decimal Engine
Código Python recalcula todos os números do zero. Se Cap Rate = NOI / Valor não bater, o erro é detectado automaticamente.
➔
2
Auditoria por IA
Specialist Prompt Audit
Uma segunda IA, com prompt de auditor financeiro sênior, revisa o relatório buscando inconsistências lógicas e premissas irreais.
➔
3
Reconciliação Cruzada
Cross-Module Reconciliation
Verifica se os dados batem entre todos os módulos: área do terreno, VGV, custos, cronogramas. Se algo diverge, volta para correção.

Guia para Desenho à Mão

Dicas práticas para reproduzir este diagrama em papel, quadro branco ou iPad

Estrutura

Desenhe 7 retângulos empilhados verticalmente, conectados por setas finas. O retângulo 4 (Orquestrador) deve ser o maior e preenchido de preto. Os blocos 2 e 3 podem ficar lado a lado.

Tipografia

Use dois tamanhos: grande para os títulos simples ("A Memória", "O Maestro") e pequeno para os termos técnicos (RAG, LLM, API). Técnico em MAIÚSCULA, simples em minúscula.

Cores

Apenas caneta preta e lápis cinza. Preencha o Orquestrador de preto com texto branco. Cinza claro para fundos dos componentes. Nenhuma outra cor.

Elementos Periódicos

Nos sub-agentes, desenhe pequenos quadrados com o símbolo do elemento (H, Fe, Au, C, Pb) — como na tabela periódica real. É a identidade visual dos agentes Ovall.

Espaço em Branco

Deixe bastante respiro entre os blocos. A elegância está na simplicidade. Se parecer apertado, aumente o papel ou reduza o texto. Menos é mais.

Anotações Laterais

Para cada bloco, escreva uma frase curta ao lado explicando o "por que" em linguagem simples. Use uma seta pontilhada conectando a anotação ao bloco.
